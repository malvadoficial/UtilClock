# Fonts folder

Put your custom font files here.

Primary segmented style (current):

- `DSEG14Modern-Regular.ttf`
- `DSEG14Modern-Bold.ttf`

Fallback segmented style:

- `DSEG7Classic-Regular.ttf`
- `DSEG7ClassicMini-Bold.ttf`

The app tries `DSEG14Modern` first, then falls back to `DSEG7`, then to Menlo.
